How to publish this site

If a host says "no index found", it means the file must be named index.html.

  1. Export or copy this code and save it on your computer as index.html (not .txt).
  2. Netlify Drop: visit app.netlify.com/drop and drag in index.html or a zip containing it.
  3. GitHub Pages: create a repo, upload index.html, then enable Pages from the repo settings.
  4. Custom domain: after it’s live, connect your domain in your host’s settings.
